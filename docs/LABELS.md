# GitHub Labels

Recommended labels:

- type:bug (red)
- type:enhancement (green)
- type:docs (blue)
- type:chore (gray)
- status:needs-triage (yellow)
- status:needs-repro (orange)
- status:blocked (black)
- scope:frontend (purple)
- scope:backend (purple)
- good first issue (light green)
- help wanted (light blue)
